Box Model y Display

Content Box

Width de la caja mas cualquier borde y/o relleno.

width: 400px;
border: 5px;
padding: 30px;

470px width
(400px (content) + 10px (border) + 60px (padding))

Border Box

Respeta el width dado a la caja, por lo que el contenido se adapta (restandole el border y padding).

width: 400px;
border: 5px;
padding: 30px;
______________+
400px width
(content - 10px (border) - 60px (padding))
